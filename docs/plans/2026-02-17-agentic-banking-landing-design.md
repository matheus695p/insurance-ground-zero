# Agentic AI in Retail Banking - Interactive Demo Page

## Overview
A React (Vite) single-page application showcasing how agentic AI transforms retail banking across all domains. QuantumBlack dark theme. 3-level navigation: Landing (domains) > Domain (use cases) > Use case (interactive demo). Deployable on Vercel.

## Premise
All analytics/ML models already exist and are deployed. The agentic AI layer connects them, automates decisions, and executes actions. Each demo shows: data source > model output > agent action > business outcome.

## Tech Stack
- React + Vite + TypeScript
- CSS Modules or Tailwind CSS
- React Router for navigation
- Framer Motion for animations
- No backend, all simulated data

## Visual Style
- QuantumBlack dark theme (dark navy/charcoal backgrounds)
- Teal/cyan accent colors
- Clean professional typography (no emojis anywhere)
- Subtle animations: typewriter effects, animated counters, fade-ins
- No marketing fluff

## 3-Level Architecture

### Level 1: Landing Page
3-row grid matching reference layout:
- Row A: Channels (Distribution Network, Customer Engagement, Marketing & Personalization)
- Row B: Business Units (Deposits, Lending, Wealth, Business Banking)
- Row C: Service Operations (Back Office, Collections, Fraud incl. KYC)

### Level 2: Domain Page
Click a domain > see all use cases as cards with brief descriptions.

### Level 3: Use Case Demo
Click a use case > full interactive simulated demo with:
- Simulated agent chat (typewriter effect)
- Fake dashboards with animated metrics
- Step-by-step process visualization
- Before/after comparisons
